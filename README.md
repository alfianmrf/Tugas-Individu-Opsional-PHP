# PHP CRUD

## Rekomendasi

- XAMPP 7.4.6

## Instalasi Database

Import Database MySQL yang bernama

```
crud.sql
```

## Menyambungkan ke PHP

Sambungkan Database MySQL ke PHP dengan cara mengubah

```
$servername = "(nama server)";
$username = "(username)";
$password = "(password)";
$dbname = "(nama database)";
```
pada file 
```
process.php
```
